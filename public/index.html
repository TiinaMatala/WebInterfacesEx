items - items Up

Information needed for creating a new product
itemId (optional)
Integer Individual id given to each item in sale
title
String The title of the item, should not include price
description
String Description of the product: color, size etc.
category
String E.g. clothes and shoes, accessories and clocks
location
String The name of the city/municipality where the product locates
images (optional)
String Max. four images of the item
askingPrice
BigDecimal The price of the item
dateOfPosting
date The date when the item was posted to the service format: date
deliveryTypeShipping
Boolean The item can be delivered to buyer
deliveryTypePickup (optional)
Boolean The item can be picked up from the seller
sellerInfoName (optional)
String The name of the seller
sellerInfoPhone (optional)
String The phone number of the seller
